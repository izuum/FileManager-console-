# uttils.FileManager-console-
uttils.FileManager (console version)

basic commands:  
  >**ll** : list of files  
  >**ls** : list of files with size  
  >**cp** : copy file  
  >**cd** : change directory  
  >**touch** : create new file  
  >**mkdir** : create new directory  
  >**cat** : file content  
  >**rename** : rename file  
  >**redir** : rename directory  
  >**rmdir** : remove directory
  >**rmfile** : remove file
  >**move** : move to directory
  >**exit** : exit
