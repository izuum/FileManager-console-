# FileManager-console-
FileManager (console version)

basic commands:  
  >**ll** : list of files  
  >**ls** : list of files with size  
  >**cp** : copy file  
  >**cd** : change directory  
  >**touch** : create new file  
  >**mkdir** : create new directory  
  >**cat** : file content  
  >**rename** : rename file  
  >**redir** : rename directory  
  >**exit** : exit
